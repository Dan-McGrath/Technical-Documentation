Introduction

What is JavaScript?

JavaScript is a cross-platform, object-orented scripting language. It is a versitale, lightweight language, primarily seen in web development. JavaScript contains a stadard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Core JavaScript can be extended for a variety of purposes either for the Frontend (client-side) or the Backend (server-side).

Console

In JavaScript the console is a panel that displays important messages, like errors, for developers. Much of the code is hidden by default so if we want to see things appear on our screen, we can print, or log, to our console directly. In JavaScript the console keyword refers to an object that we can use in our code. Keywords are words that developers have built into the language. By doing this the computer will recognize them out of the box and will treat them differently. One method that is built into the console onject is the .log() method. When we write console.log(); , whatever we put inside the parentheses will get printed, or logged, to the console. Input:
console.log('Hello World')
Output:
Hello World This helps us see what our code is doing. We can use this to see what a variables value is, or to help with error-handling as well as many other use cases. We add a semicolon to the end of our code to indicate the end of the line, or statement. While JavaScript will usually run as intended without it, it is common practice to use it for the small cases that they are needed.
